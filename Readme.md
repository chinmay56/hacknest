## Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/hacknest.git
```

2. Open the project
```bash
cd hacknest
```

3. Start the development server
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

4. Open in browser
```
http://localhost:8000
```
